# Currency-rate
Приложение "Курс Валют", разработанное в рамках выпускной работы
по программе дополнительного образования Цифровая кафедра "Frontend-разработчик"

Главная задача приложения «Валюты»
Веб-приложение позволяет пользователю узнать следующую информацию о валютах, а именно:
1)	название валюты;
2)	актуальный курс;
3)	динамику изменения курса по сравнению с предыдущим днем;
4)	рассчитать сумму определенной валюты в рублях (конвертация валюты в рубли).
